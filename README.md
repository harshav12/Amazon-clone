# Amazon Clone (HTML & CSS)

A static front-end clone of the Amazon homepage built using HTML and CSS.  
This project was created to practice layout design, CSS styling, and page structuring by recreating a real-world e-commerce UI.

---

## Features
- Amazon-style navigation bar with logo, search box, and cart icon
- Header panel with category links
- Hero section with promotional banner
- Product grid section displaying multiple product categories
- Footer section with informational links
- Hover effects for interactive UI elements

---

## Tech Stack
- HTML5
- CSS3
- Font Awesome (icons)

---

## Project Type
- Static website
- Front-end UI clone
- No JavaScript functionality

---

## Limitations
- Not fully responsive across different screen sizes
- Static content (no backend or dynamic data)
- Search bar and buttons are non-functional
- Designed primarily for desktop view

---

## Learning Outcomes
- Improved understanding of HTML page structure
- Hands-on experience with CSS Flexbox and layout design
- Practice using icons and images in UI design
- Recreating real-world website layouts using pure HTML & CSS

---

## How to Run
1. Clone or download the repository
2. Open `index.html` in any modern web browser
